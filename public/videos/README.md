# Video for Learn tab

To show **"Your First Investment: Growing Money for the Future"** on the website:

1. **Convert your video to MP4** (recommended). Browsers support MP4 (H.264) best. You can convert `.mov` to `.mp4` using QuickTime (File → Export) or any converter.

2. **Save the file in this folder** with one of these names:
   - `first-investment.mp4` (preferred)
   - or `first-investment.mov`

3. The Learn page will automatically use it at `/learn/video-mvp1`.

Path in your repo: **`Sarasage/public/videos/first-investment.mp4`**
