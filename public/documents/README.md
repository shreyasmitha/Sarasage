# Governance & data framework PDF

To show the framework on the **Data & governance** page (`/governance`):

1. Add your PDF file to this folder with this **exact filename**:
   - **`governance-data-framework.pdf`**

2. Full path in your repo:
   ```
   Sarasage/public/documents/governance-data-framework.pdf
   ```

3. Commit and push (or redeploy on Vercel). The page embeds the PDF and offers “Open in new tab” and “Download”.

If the file is missing, the iframe will be empty until you add the PDF.
